# Python仓库管理系统

## 核心功能
- 登录注册
- 产品分类管理
- 仓库管理
- 入库管理
- 出库管理
- 员工管理
- 进货管理
- 出货管理
- 借用管理
- 归还管理

## 适用场景
- 企业仓库管理
- 仓储物流中心
- 商品库存跟踪
- 供应链管理

## 技术特点
- 后端框架：Python Flask
- 前端框架：vue
- 数据库：mysql
- 开发工具：Pycharm、navicat
- 运行环境：Python3.0以上、mysql8.0
